More info

This peice is by Bridget Riley. Riley is a pioneer in the OP art movement. Serpentine was created as a benefit print and showcased at the Serpentine Gallery in London. It is a color screenprint on woven paper. I have recreated this using CSS html You are viewing the end result of different styles and tags used to recreate Serpentine by Bridget Riley
